# Unreal Engine 5.7 – Third Person Locomotion System

A high-quality **third-person locomotion system** built in **Unreal Engine 5.7**, focused on delivering smooth, realistic, and weighty movement inspired by modern AAA games like *Resident Evil*.

---

## 🧙 Features

**8-Way Locomotion**
  Full directional movement (forward, backward, strafing) with smooth blending.

**Orientation & Stride Warping**
  Dynamic animation adjustment for natural movement and accurate foot alignment.

**Foot Placement + Advanced Leg IK**
  Stable and realistic foot contact on uneven surfaces.

**Character Lean System**
  Subtle body leaning during movement for added realism and responsiveness.

**Aim Offset System**
  Smooth upper-body aiming independent from lower-body movement.

**Heavy Movement Style**
  Inspired by *Resident Evil* — grounded, weighty, and immersive locomotion.

---

## 🛠️ Tech Stack

* **Engine:** Unreal Engine 5.7
* **System:** Blueprint-based (expandable to C++)
* **Animation:** State Machines, Blend Spaces, IK systems
* **Input:** Enhanced Input System

## 📂 Project Structure

* `Characters/` → Character setup and logic
